## トラブルシューティング

### Q1: 途中でブロックされた
**A:** 遅延時間を増やす

`update_from_list.py` の設定を編集：
```python
self.DELAY_HORSE_MIN = 3.0  # 2.0 → 3.0
self.DELAY_HORSE_MAX = 8.0  # 5.0 → 8.0
self.BATCH_COOLDOWN_MAX = 120  # 60 → 120
```

### Q2: PCがスリープしてしまう
**A:** 電源設定を変更

Windows:
1. 設定 → システム → 電源とスリープ
2. スリープ: なし
3. ディスプレイの電源を切る: なし（推奨）

### Q3: エラーが頻発する
**A:** ネットワーク確認

- 有線LAN推奨
- WiFi使用時は安定性確認
- VPN使用時は切断を試す

### Q4: 処理が遅すぎる
**A:** これは正常です

- 1レースあたり2-5分は正常
- サーバー負荷対策のため意図的に遅くしています
- 高速化したい場合は遅延時間を短縮（非推奨）

- 夜間・早朝の実行推奨
- エラー頻発時は一時停止
