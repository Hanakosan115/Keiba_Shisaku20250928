# Phase 9 競馬予想AI - 本番運用対応版

## 概要

Phase 9モデルは、Phase 8の240.6% ROIを維持しつつ、ラップタイムと調教データを追加した改良版です。2025年のバックテストで**236.5% ROI**、期待値2.0以上の戦略では**408.9% ROI**を達成しています。

## 主要機能

### 1. データ収集の完全化

- **2020-2025年の完全データ**: 466,746レコード
- **ラップタイム**: 97.2%の取得率
- **調教データ**: 96.3%の取得率
- **データファイル**: `data/main/netkeiba_data_2020_2025_complete.csv`

### 2. Phase 9モデルの特徴

**特徴量**: 63個（54基本 + 9新規）

**新規特徴量**:
- `lap_count`: ラップ数
- `first_3f_avg`: 序盤3F平均
- `last_3f_avg`: 終盤3F平均
- `pace_variance`: ペース分散
- `pace_acceleration`: ペース加速度
- `training_rank_numeric`: 調教ランク（S=5, A=4, B=3, C=2, D=1）
- `pace_fast`, `pace_medium`, `pace_slow`: ペース分類

**重要特徴量 Top 5**:
1. `avg_last_3f`: 358.0
2. `days_since_last_race`: 278.0
3. `trainer_win_rate`: 274.0
4. `time_consistency`: 269.0
5. `mother_father_top3_rate`: 257.0

### 3. バックテスト結果（2025年）

- **対象**: 2,375レース
- **的中率**: 42.6% (1,011/2,375)
- **ROI**: 236.5% (全レース購入)

**期待値ベース戦略**:
| 期待値閾値 | 購入レース数 | 的中率 | ROI |
|-----------|-------------|--------|------|
| ≥ 1.0 | 1,583 | 40.5% | 293.8% |
| ≥ 1.2 | 1,355 | 39.6% | 319.0% |
| ≥ 1.5 | 1,094 | 37.2% | 347.2% |
| ≥ 2.0 | 809 | 35.4% | **408.9%** |

**1番人気との比較**:
- 1番人気: 的中率 33.3%, ROI 79.0%
- Phase 9: 的中率 42.6%, ROI 236.5%

### 4. GUI v2の本番対応機能

#### 主要改善点

**a) 完全なレース情報抽出**
```python
- 競馬場名 (track_name)
- 距離 (distance)
- コースタイプ (course_type: 芝/ダート)
- 馬場状態 (track_condition: 良/稍重/重/不良)
- 枠番 (frame number)
```

**b) 正確な特徴量計算**
- 実際のレース条件を使用して特徴量を計算
- `calculate_horse_features_dynamic()`に全パラメータを渡す
- コース適性、馬場適性、枠適性を正確に反映

**c) 過去成績の表示**
- 各馬の戦績（例: "15戦3勝5着内"）
- 直近5走の着順
- データ有無を✓/✗で明示

**d) CSV エクスポート**
- 予測結果を保存可能
- ファイル名形式: `prediction_[レースID]_[タイムスタンプ].csv`

**e) カラー表示**
- 上位3頭をハイライト
- 成功メッセージを緑、エラーを赤で表示
- 期待値1.2以上をオレンジ、2.0以上を緑で強調

**f) 進捗表示**
- プログレスバーで処理状況を表示
- ステータスバーで現在の処理内容を表示

## 使用方法

### 1. GUIの起動

```bash
python keiba_prediction_gui_v2.py
```

### 2. レース予想の実行

1. レースIDを入力（例: 202501040101）
   - 形式: YYYYMMDDPPNN
   - YYYY: 年, MM: 月, DD: 日, PP: 場所, NN: レース番号

2. 「予想開始」ボタンをクリック

3. 結果の確認:
   - 各馬の勝率予測、複勝予測、期待値
   - 推奨馬券（本命、対抗、単穴）
   - 期待値ベース推奨馬

4. 必要に応じて「結果をCSV保存」で保存

### 3. 推奨投資戦略

**保守的戦略**: 期待値 ≥ 2.0
- ROI: 408.9%
- 的中率: 35.4%
- 年間約800レース

**積極的戦略**: 期待値 ≥ 1.5
- ROI: 347.2%
- 的中率: 37.2%
- 年間約1,100レース

**バランス戦略**: 期待値 ≥ 1.2
- ROI: 319.0%
- 的中率: 39.6%
- 年間約1,350レース
