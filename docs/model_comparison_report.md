## 総合評価

### 🥇 最優秀モデル: **脚質特徴量モデル (28次元)**

**選定理由:**
1. **最高の回収率**: 173.7% (ワイド_1-2)
2. **バランスの良い性能**: 的中率と回収率の両立
3. **実用性**: 訓練・予測が高速
4. **穴馬発見能力**: オッズに依存せず高配当馬を捕捉

### ベースラインモデルの強み
- 的中率重視なら最適
- シンプルで理解しやすい
- オーバーフィッティングのリスク低

### 次世代統合モデルの課題

#### 問題点の分析

1. **過学習の可能性**
   - 42次元は特徴量が多すぎる可能性
   - 訓練データに過度に最適化
   - 汎化性能が低下

2. **ノイズ特徴の混入**
   - 追加した14次元の一部が実は予測に寄与していない
   - むしろノイズとして機能している可能性

3. **特徴量の相関**
   - track_aptitude と avg_rank は相関が高い
   - 距離適性と競馬場適性も重複する情報
   - 多重共線性の問題

4. **展開予想の精度不足**
   - 簡易的なペース判定ロジック
   - 実際のペース展開との乖離

#### 改善案

**A. 特徴量の削減（次元削減）**
1. 特徴量重要度を分析
2. 重要度の低い特徴を削除
3. 30次元程度に絞る

**B. より高度な展開予想**
1. 過去のペース展開データを学習
2. 騎手の作戦傾向を反映
3. 開催週による馬場変化を考慮

**C. 正則化の強化**
1. LightGBMの正則化パラメータ調整
2. min_child_samplesの増加
3. max_depthの削減

**D. アンサンブル学習**
1. 脚質モデル(28次元) + 一部の適性特徴
2. 複数モデルの予測を組み合わせ
