## 🥇 提案1: Phase 14完全開発（最優先）

### 概要

Phase 14複勝モデルを完全に開発し、Phase 13（単勝）との比較・実運用準備までを完了させる。

### 詳細工程

#### Phase 1: 準備（30分）
1. カラム名マッピング作成
2. データベース前処理
3. スクリプト準備

#### Phase 2: 特徴量計算（4-8時間）
- **バックグラウンド実行**（他作業と並行可能）
- 283,925レコードの特徴量計算
- 10,000レースごとにチェックポイント保存

#### Phase 3: モデル訓練（2-3時間）
1. 単勝モデル訓練（Phase 14用）
2. **複勝モデル訓練**（新規）
3. 交差検証（5-Fold）
4. モデル保存

#### Phase 4: バックテスト検証（1-2時間）
1. 2024年全レースでバックテスト
2. 確率帯別分析
3. Phase 13との比較

#### Phase 5: Phase 13再検証（1-2時間）
1. クリーンバックテストで真の性能評価
2. 219%との比較
3. 実用可能性判定

#### Phase 6: 文書化（1-2時間）
1. 開発レポート作成
2. 性能比較レポート
3. 実運用マニュアル更新

### 総所要時間

**合計**: 8-14時間
- 準備: 0.5時間
- 特徴量計算: 4-8時間（バックグラウンド）
- モデル訓練: 2-3時間

- 文書化: 1-2時間

### 期待される成果

#### 意思決定材料
- ✅ Phase 13は実用可能か？
- ✅ Phase 14は Phase 13より優秀か？
- ✅ 単勝と複勝、どちらで運用すべきか？

## 🥈 提案2: Phase 13全確率帯クリーン検証

### 概要

Phase 13モデルで2024年全レースを**リーク防止モード**で再検証し、
全10確率帯の真の回収率を算出する。

### 詳細工程

#### 1. 準備（30分）
- カラム名問題解決（提案1と共通）

#### 2. Phase 13クリーンバックテスト実行（1-2時間）
- 2024年全レース（約3,200レース）
- リーク防止保証（current_date=レース日付）
- 全10確率帯で分析

#### 3. 分析・レポート作成（30分-1時間）
- 確率帯別回収率
- 219%との比較
- 実用可能性判定

### 総所要時間

**合計**: 2-3時間

### 期待される成果

#### 真の性能データ

**従来（リークあり）**:
```
40-50%確率帯: 219.1%回収率
35-40%確率帯: 166.2%回収率
全確率帯: 100%超
```

**真の性能（予想）**:
```
40-50%確率帯: 95-110%回収率
35-40%確率帯: 90-105%回収率
一部確率帯: 100%超
```

#### 意思決定

- ✅ Phase 13は実用可能か？
- ✅ どの確率帯が最適か？
- ✅ 期待収益はいくらか？

#### 実運用への道筋

**回収率が80-100%の場合**:
- 実用可能
- ただし慎重に運用
- 少額から開始

**回収率が100-120%の場合**:
- 極めて優秀
- 積極的運用可能
- Phase 14開発の価値も高い

**回収率が80%未満の場合**:
- 実用困難
- Phase 14開発に注力

### 価値

**★★★★☆（高い）**

**理由**:
1. Phase 13の真の実力が判明
2. 実運用の可否が明確に
3. 比較的短時間で完了
4. Phase 14開発の判断材料

### 提案1との関係

**Phase 14開発（提案1）の一部として実施可能**
- Phase 14開発時にPhase 13も同時検証
- 一石二鳥

## 🥉 提案3: 2020-2025完全バックテスト

### 概要

Phase 13モデルで2020-2025年の**全6年分**をバックテストし、
長期的な性能安定性を評価。

### 詳細工程

#### 1. 準備（30分）
- カラム名問題解決

#### 2. 6年分バックテスト実行（2-4時間）
- 2020年: 約48,000レース
- 2021年: 約48,000レース
- 2022年: 約47,000レース
- 2023年: 約48,000レース
- 2024年: 約47,000レース
- 2025年: 約46,000レース
- **合計: 約284,000レース**

#### 3. 分析（30分-1時間）
- 年別回収率
- 確率帯別の経年変化
- 月別・季節別分析
- トレンド分析

### 総所要時間

**合計**: 3-5時間

### 期待される成果

#### 長期性能データ

```
年別回収率:
2020年: 88.5%
2021年: 92.3%
2022年: 86.7%
2023年: 91.2%
2024年: 89.8%
2025年: 90.5%

6年平均: 89.8% ± 2.1%
```

#### 重要な発見

- ✅ モデルの安定性
- ✅ 性能劣化の有無
- ✅ 経年変化のパターン
- ✅ 再訓練の必要性

#### 実運用への示唆

**安定している場合**:
- 長期運用可能
- 信頼性が高い

**劣化トレンドがある場合**:
- 定期的な再訓練が必要
- モデル更新戦略の策定

### 価値

**★★★★☆（高い）**

**理由**:
1. 長期的な信頼性評価
2. 実運用の安心材料
3. 大量データによる精度向上

### 注意

**計算時間が長い**:
- 284,000レースの処理
- 2-4時間の連続実行

**ディスク容量**:
- 結果CSVが大きい（約50-100MB）
