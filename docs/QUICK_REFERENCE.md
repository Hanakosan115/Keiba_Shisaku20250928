## 📅 週次運用フロー

### 毎週月曜（前週のレース結果を反映）

```
1. データ取得（元ツール）
   → HorseRacingAnalyzerで最新データをスクレイピング

2. データ更新
   → py update_main_data.py
   → 新しいCSV/JSONのパスを入力

3. データ検証（GUIツール）
   → keiba_analysis_tool.py を起動
   → データ管理タブで「データ統計を更新」
   → 未来データ警告がないか確認

4. 完了！
```

### 毎週金曜（週末レース予測）

```
1. race_idを調べる
   → netkeibaで週末レースのrace_idを確認

2. 一括予測
   → py predict_weekend_races.py
   → race_idを入力

3. 結果CSVを確認
   → weekend_predictions_YYYYMMDD_HHMMSS.csv

4. 馬券購入（任意）
```

## 🗓️ 月次運用フロー

### 月初（モデル再訓練）

```
1. データ更新（週次フローと同じ）

2. モデル再訓練（GUIツール）
   → モデル訓練タブ
   → 終了年月を最新に更新（例: 2024年12月）
   → 「訓練開始」クリック

3. バックテスト（GUIツール）
   → バックテストタブ
   → テスト年を最新に設定
   → 「バックテスト実行」クリック
   → 回収率を確認（80%以上が目安）

4. 結果を記録
   → 月次で回収率の推移をExcelなどで管理
```

## 🔧 トラブルシューティング

### 未来データ警告が出る
```
→ データ管理タブで「データクリーニング」を実行
→ 元データの取得期間を確認（今日まで）
```

### race_idが見つからない
```
→ データに含まれているか確認
→ py get_test_race_ids.py で利用可能なrace_idを表示
→ 2024年のrace_idを使用（2025年データは未来データとして除外済み）
```

### モデル訓練が失敗する
```
→ データ検証で品質スコアを確認（90点以上が目安）
→ 訓練期間が1年以上あるか確認
→ エラーメッセージを確認
```

### 予測が外れ続ける
```
→ バックテストで精度を確認
→ 確信度フィルタを使う（上位10%など）
→ 複数戦略を組み合わせる
→ 馬券購入額を調整（レース予算の5%以下推奨）
```

## 💡 Tips

### race_idの調べ方
```
netkeibaのレースページURL末尾12桁
例: https://race.netkeiba.com/race/result.html?race_id=202406030812
    → race_id = 202406030812
```

### 2024年のテスト用race_id例
```
重賞レース:
  202401010111  - TVh杯(3勝)
  202401010411  - クイーンステークス(GIII)
  202401010611  - エルムステークス(GIII)

11月レース:
  202403030101 ～ 202403030105

12月レース:
  202406050201 ～ 202406050205
```

### 確信度フィルタの使い方
```
バックテストタブで設定:
  - なし: 全レースをテスト（網羅的）
  - 上位10%: 超厳選（少数精鋭）
  - 上位20%: バランス（推奨）
  - 上位30%: やや広め

実運用では上位10-20%を推奨
```

### 投資額の目安
```
月間予算を決める（例: 10,000円）
  → 週末2日×4週 = 8開催
  → 1開催あたり 1,250円
  → 確信度の高いレース3レースに絞る
  → 1レースあたり 400円程度

複数戦略に分散:
  - ワイド_1-3: 200円
  - 3連複_BOX5頭: 200円（10点×20円）
```

## 📊 記録の取り方（推奨）

Excelで以下を記録:

| 日付 | race_id | レース名 | 戦略 | 購入額 | 的中 | 払戻 | 収支 | 累計 |
|------|---------|---------|------|--------|------|------|------|------|
| 2024-12-01 | 202406050201 | 2歳未勝利 | ワイド_1-3 | 100 | ✓ | 350 | +250 | +250 |
| 2024-12-01 | 202406050202 | 2歳未勝利 | 3連複_BOX5 | 1000 | ✗ | 0 | -1000 | -750 |

月末に集計して回収率を計算:
```
回収率 = 総払戻 / 総購入額 × 100
```

## ⚠️ 重要な注意事項

1. **あくまで娯楽**: 馬券は娯楽です。生活費を使わないでください
2. **予算厳守**: 月間予算を決めて絶対に超えない
3. **自己責任**: 予測は100%的中しません。最終判断は自己責任で
4. **熱くならない**: 負けを取り返そうとしない
5. **記録を残す**: 購入記録を残して振り返る
