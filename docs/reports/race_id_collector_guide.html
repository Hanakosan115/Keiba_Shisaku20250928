🔧 収集手順

ステップ1: カレンダーページを開く
https://race.netkeiba.com/top/calendar.html?year=2025&month=9
ステップ2: 収集したい日付をクリック（例: 9月6日）
ステップ3: レース一覧ページで F12 を押して開発者ツールを開く
ステップ4: Console タブをクリック
ステップ5: 以下のコードをコピーして、Consoleに貼り付けて Enter
const raceIds = new Set(); document.querySelectorAll('a[href*="race_id="]').forEach(a => { const match = a.href.match(/race_id=(\d{12})/); if (match) raceIds.add(match[1]); }); Array.from(raceIds).sort().forEach(id => console.log(id));
ステップ6: 出力されたレースIDをコピー（全て選択して Ctrl+C）
ステップ7: race_ids.txt に貼り付け
ステップ8: 統合GUIツールでデータ更新を実行
✅ 完了！
各週末について同じ手順を繰り返せば、完璧なデータベースが完成します。

⏱️ 時間の見積もり

  • 各週末のレースID収集: 約3分
  • 14週末 × 3分 = 約42分
  • 自動スクレイピング: 約10-15分（自動）
  • 合計: 約1時間で完璧なデータベース！

💡 ショートカット

複数日をまとめて収集:
1つの日付で上記のコードを実行したら、すぐに別の日付のタブに切り替えて同じコードを実行。
全てのレースIDを1つの race_ids.txt にまとめて貼り付ければ、一気に処理できます。
