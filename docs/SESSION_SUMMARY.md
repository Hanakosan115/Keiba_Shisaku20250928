### Ver 3.0: 次世代統合モデル（42次元）← 訓練中！

**追加した特徴量（+14次元）**:

#### A. 脚質特徴の拡張（+2次元）
7. `position_change_ability`: 位置取り変化能力
8. `agari_stability`: 上がり3Fの安定性

#### B. コース適性（+6次元）

12. `distance_experience`: 同距離帯での経験数
13. `turn_aptitude`: 回り方向適性（左/右）
14. `turn_experience`: 同回り方向での経験数

#### C. 展開予想（+4次元）
15. `is_high_pace`: ハイペースフラグ
16. `is_slow_pace`: スローペースフラグ
17. `pace_advantage`: ペース適性スコア
18. `escape_competition`: 逃げ競合度

#### D. 回り方向（+2次元）
19. `is_left_turn`: 左回りフラグ
20. `is_right_turn`: 右回りフラグ

**実装した高度な機能**:

1. **レース展開予想**
   ```python
   # 出走馬の脚質分布から予想ペースを判定
   if 逃げ馬 >= 2頭:
       pace = "ハイペース"  # 逃げ争い
   elif 逃げ馬 == 1頭 and 先行馬 <= 2頭:
       pace = "スローペース"
   else:
       pace = "平均ペース"
   ```

2. **ペース適性判定**
   ```python
   if ハイペース and 馬の脚質 in [差し, 追込]:
       pace_advantage = +1  # 有利
   elif スローペース and 馬の脚質 in [逃げ, 先行]:
       pace_advantage = +1  # 有利
   else:
       pace_advantage = -1  # 不利
   ```

3. **トラックバイアス対応**
   - 競馬場マスタデータで回り方向を管理
   - 左回り/右回りごとの適性を計算
   - 馬ごとの得意な回り方向を特定

4. **4コーナー分のPassage解析**
   - 従来: "5-5"（序盤-終盤のみ）
   - 拡張: "11-12-15-8"（全コーナーの位置）
   - 位置取りの変化を定量化

**期待される効果**:
- 3連複的中率: 9.0% → **12%以上**
- 全戦略で回収率: **140%以上**
- オッズ依存からの脱却
- トラックバイアスの活用

## 技術的な改善

### パフォーマンス最適化

**問題**: CSVの繰り返し読み込みで訓練が非常に遅い

**解決策**:
```python
# Before: 毎回CSVを読み込む
past_results = get_horse_past_results_from_csv(horse_id, race_date)

# After: DataFrameを直接使用
past_races = df[(df['horse_id'] == horse_id) &
                (df['date_parsed'] < race_date)]
```

**効果**: 訓練時間が **10分以上 → 数分** に短縮

## 主要な発見

### 1. Passageデータの秘密
- 一部のレースで4つの位置情報がある
- "11-12-15-8" = 各コーナーでの位置
- **セクショナル分析が可能**

### 2. 脚質特徴の効果
- 的中率は下がったが回収率は向上
- **穴馬の発見能力が向上**
- オッズだけでは見抜けない適性馬を特定

### 3. トラックバイアスの重要性
- 競馬場ごとに脚質の有利不利がある
- 回り方向（左/右）で適性が分かれる
- 開幕週 vs 後半週で展開が変わる
