## 💡 今回の教訓

### 1. Unicode/エンコーディング問題
- Windows コンソール（cp932）では絵文字・特殊文字が使えない
- 解決策: 絵文字を避ける、またはログファイルにリダイレクト

### 2. インポートの事前確認
- 存在しない関数のインポートはスクリプト全体を停止させる
- 解決策: `dir(module)` で確認、またはコードレビュー

### 3. データ構造の検証
- 取得したデータは必ず構造を確認すべき
- 解決策: サンプルデータの出力とチェック

### 4. バックグラウンドタスクのログ
- 詳細なログがないと問題の特定が困難
- 解決策: try-except で全エラーをキャッチしログ出力
