## 🎯 確率帯選択ガイド

### あなたに最適な確率帯は？

#### Q1: あなたの投資スタイルは？
- **安定志向** → 30%以上（戦略1）
- **バランス志向** → 15-40%（戦略2）
- **高リターン志向** → 40-50%集中（戦略3）
- **ギャンブル志向** → 10%以下（戦略4）

#### Q2: どのくらい待てますか？
- **毎週的中がほしい** → 30%以上（的中率50%以上）
- **月に数回でOK** → 15-30%（的中率30-40%）
- **年に数回の大当たり狙い** → 10%以下（的中率20%以下）

#### Q3: 資金管理の自信は？
- **自信なし** → 30%以上（連敗が少ない）
- **普通** → 15-40%（適度な連敗）
- **自信あり** → 全確率帯OK

## ✅ 実運用での注意点

### 1. 確率帯の識別方法
GUIで「勝率予測」列を確認:
```
勝率予測    確率帯
0.187      → 15-20%
0.425      → 40-50%
0.125      → 10-15%
```

### 2. 複数確率帯の組み合わせ
- 同じレースで複数の確率帯に該当する馬がいる場合
- より高回収率の確率帯を優先
- またはリスク分散のため両方購入

### 3. オッズ変動への対応
- GUIのオッズは取得時点のもの
- 実際の購入時に大きく変動している場合は見送り
