## 💡 なぜ複勝は失敗したのか？

### 理由1: 払戻の構造的な低さ
- 複勝平均払戻: 138-230円（確率帯により変動）
- 単勝平均払戻: 推定800-900円
- **複勝は本質的に低配当**

### 理由2: モデルの目的関数の不一致
Phase 13モデルは**勝率（1着）**を予測するよう学習:
- 目的変数: 1着 = 1, それ以外 = 0
- 最適化: 単勝回収率

複勝は**複勝率（1-3着）**を予測すべき:
- 目的変数: 1-3着 = 1, それ以外 = 0
- 最適化: 複勝回収率

**異なる目的には異なるモデルが必要**

### 理由3: 的中率と払戻のトレードオフ
複勝の期待値:
```
期待値 = 的中率 × 平均払戻 - 100円

例: 50%+確率帯
  = 39.5% × 138円 - 100円
  = 54.5円 - 100円
  = -45.5円 ❌
```

単勝の期待値:
```
例: 15-20%確率帯
  = 17% × 850円 - 100円
  = 144.5円 - 100円
  = +44.5円 ✅
```
