## ⚠️ 制限事項

### 1. 理論計算の精度限界

**複勝オッズの推定誤差**:
```
推定式: 複勝オッズ = 単勝オッズ / 2.5
実際: 人気馬は /3, 穴馬は /2 程度になることが多い
誤差: ±20-30%
```

**ワイドオッズの推定誤差**:
```
推定式: ワイドオッズ = (単勝1 + 単勝2) / 3
実際: 組み合わせや人気により大きく変動
誤差: ±30-50%
```

### 2. サンプル数の不足

**統計的有意性の基準**:
- 最低必要サンプル数: 100レース
- 推奨サンプル数: 500レース以上
- 95%信頼区間の精度: サンプル数の平方根に依存

**実施したサンプル数**:
- 15-20%確率帯（複勝等）: 31レース ← 不足
- WIN5: 10日間 ← 大幅に不足

**信頼性への影響**:
- 31レースでの95%信頼区間: 幅が広すぎる
- 偶然の要素が大きく影響
- 再現性が保証できない

### 3. データ取得の制限

**取得できなかったデータ**:
- 複勝の実払戻
- ワイドの実払戻
- 馬連・馬単・3連複・3連単の実払戻
- WIN5の実配当
- 過去の配当分布データ

**原因**:
- `get_race_from_database()` が払戻情報を返さない
- データベースに払戻情報が保存されていない
- スクレイピングによる取得を実施しなかった
