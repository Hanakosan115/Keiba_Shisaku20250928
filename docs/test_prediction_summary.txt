【使用方法】
週末が近づいたら（木曜〜金曜）：
  py predict_this_weekend.py

レース当日：
  1. プログラムを実行
  2. 表示されるレース一覧から予想したいレースを選択
  3. 予算を入力（例: 10000円）
  4. Value Betting分析に基づく推奨ベットが表示されます
  5. 推奨に従って馬券を購入

【Value Betting戦略】
- 予想順位とオッズから期待値を計算
- Value（期待値 - 1）が正の馬を抽出
- 予算をKellyルールで最適配分
- 長期的な利益を最大化
