# 未来レース予測ガイド
## 明日のマイルCSを予測する方法

---

## 🎯 概要

このガイドでは、**明日のマイルチャンピオンシップ（マイルCS）** など、未来のレースを予測する方法を説明します。

### 仕組み

```
1. 過去データで訓練したモデル
   ↓
2. 出馬表データ（明日のレース情報）
   ↓
3. 予測実行
   ↓
4. 推奨馬券の提示
```

---

## 📋 必要な準備

### 1. モデルの訓練（済）
- GUIツールで2020-2025年のデータでモデル訓練
- `race_prediction_model_202508.pkl` が作成済み

### 2. 出馬表データの取得（これから）
- netkeibaから明日のレースの出馬表を取得
- 馬番、馬名、騎手、オッズなどの情報

### 3. 必要なライブラリ
```bash
pip install requests beautifulsoup4 lxml
```

---

## 🚀 実行手順（3ステップ）

### Step 1: 出馬表データを取得

```bash
cd C:\Users\bu158\Keiba_Shisaku20250928
python fetch_shutuba_data.py
```

**入力内容:**
```
race_id: 202506030811
（例: 2025年11月23日 京都11R マイルCS）
```

**race_idの調べ方:**
1. netkeiba.com を開く
2. レースカレンダーから明日のレースを探す
3. レースページのURLを確認
   ```
   https://race.netkeiba.com/race/shutuba.html?race_id=202506030811
   → race_id = 202506030811
   ```

**出力:**
- `shutuba_202506030811_YYYYMMDD_HHMMSS.csv`
- 出走馬のデータ（馬番、馬名、騎手、オッズなど）

---

### Step 2: 予測を実行

```bash
python predict_with_shutuba.py
```

**入力内容:**
```
ファイル名（または番号）: 1
（Step 1で作成したshutubaファイルを選択）
```

**処理内容:**
1. 出馬表データを読み込み
2. 訓練済みモデルを読み込み
3. 各馬のスコアを計算
4. 着順を予測
5. 推奨馬券を生成

**出力:**
- 予測着順（TOP10）
- 推奨馬券（ワイド、3連複、馬連、3連単）
- `prediction_202506030811_YYYYMMDD_HHMMSS.csv`
- `prediction_202506030811_YYYYMMDD_HHMMSS.txt`

---

### Step 3: 結果を確認して馬券購入

予測結果のテキストファイルを開いて確認：
```
予測着順 TOP5:
1. 3番 ソングライン (ルメール)
2. 7番 レシステンシア (武豊)
3. 1番 グランアレグリア (川田将雅)
4. 5番 インディチャンプ (福永祐一)
5. 12番 ダノンスコーピオン (横山典弘)

推奨馬券:
ワイド 1-3: 3-1 (100円)
3連複 BOX5: 3-7-1-5-12 (1,000円)
馬連 1-2: 3-7 (100円)
```

**購入判断:**
- 確信度を参考に購入額を決定
- 予算を守る（月間予算の範囲内）
- 複数戦略に分散

## 🔧 トラブルシューティング

### Q1: 出馬表が取得できない

**エラー:**
```
エラー: 出馬表が見つかりません
```

**原因:**
- race_idが間違っている
- レースがまだ確定していない
- netkeibaのページ構造が変更された

**解決策:**
1. race_idを再確認
2. レース確定を待つ（通常は数日前に確定）
3. 手動入力モードを使用:
   ```bash
   python predict_future_race.py
   ```

---

### Q2: オッズが取得できない

**症状:**
```
Odds: None
```

**原因:**
- オッズ発表前（レース数日前）

**解決策:**
- デフォルトオッズで予測（暫定）
- オッズ発表後に再度取得

---

### Q3: モデルが見つからない

**エラー:**
```
モデルファイルが見つかりません
```

**解決策:**
1. GUIツールでモデル訓練
   ```bash
   python keiba_analysis_tool.py
   ```
2. モデル訓練タブで訓練実行
3. `race_prediction_model_*.pkl` が作成される

## 🎯 期待できる精度

### バックテスト結果（2024-2025年）

| 戦略 | 的中率 | 回収率 |
|------|--------|--------|
| ワイド 1-3 | 24.2% | 87.7% |
| 3連複 BOX5頭 | 35.3% | 84.8% |
| 馬連 1-2 | 30.4% | 83.5% |

**注意:**
- 回収率100%未満 = 損失
- 80%台 = 損失を最小化レベル
- 競馬は控除率25%のため、理論上の限界あり

**現実的な目標:**
- 的中を楽しむ（エンターテイメント）
- 損失を最小化する
- 長期的に80%以上の回収率を維持

---

## ⚠️ 重要な注意事項

### 1. 予算管理
- 月間予算を決める（例: 10,000円）
- 週間予算に分割（例: 2,500円/週）
- 絶対に超えない

### 2. 自己責任
- 予測は100%的中しない
- 最終判断は自己責任
- 生活費を使わない

### 3. 娯楽として楽しむ
- 競馬は娯楽
- 儲けを期待しすぎない
- プロセスを楽しむ

### 4. 記録を残す
- 購入馬券を記録
- 的中・不的中を記録
- 月次で振り返る
