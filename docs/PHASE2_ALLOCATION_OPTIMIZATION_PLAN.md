## 🎯 最適化方針

### 基本原則
1. **ROI重視**: 高ROI馬券種に厚く配分
2. **リスク分散**: 的中率も考慮（高ROIでも的中率極低は避ける）
3. **合計金額**: 勝率が高いほど投資額を増やす（確信度に応じた配分）

### ROI vs 的中率のバランス

期待値 = 投資額 × ROI / 100

例: 単勝300円 × ROI 298% = 期待回収 894円（+594円）

## 🎯 実装方針

### 1. 合計金額の統一
- 高確信帯（50%+, 40-50%, 30-40%, 20-30%, 10-20%）: **600-700円**
- 見送り帯（<10%）: **100円 or 見送り**

### 2. ROI優先順位
各勝率帯で最もROIが高い馬券種に厚く配分

### 3. 表示の改善
```
【推奨買い目】勝率に応じた最適パターン

  ─ 本命レース（勝率50%以上）─
  単勝: 5番 400円 ← 77.4%的中/ROI 298% ★メインベット
  3連複: 5-3-8 200円 ← 16.0%的中/ROI 449% (高配当狙い)
            ↑複勝率上位3頭
  複勝: 5番 100円 ← 91.3%的中/ROI 143% (保険)
  → 合計700円 / 期待収支 +198円
```
