## 追加する特徴量（全12次元追加予定）

### 【A. 展開予想関連】(+6次元)

#### 1. race_predicted_pace (レース全体の予想ペース)
```python
# 出走馬の脚質分布から計算
逃げ馬率 = 逃げ馬の数 / 出走頭数
先行馬率 = 先行馬の数 / 出走頭数

if 逃げ馬の数 >= 2:
    predicted_pace = "ハイペース"  # 逃げ争い
elif 逃げ馬の数 == 1 and 先行馬率 < 0.3:
    predicted_pace = "スローペース"  # 逃げ1頭、先行少ない
else:
    predicted_pace = "平均ペース"
```

#### 2. escape_competition (逃げ競合度)
```python
# 同じレースに何頭の逃げ馬がいるか
逃げ馬数 = レース内の逃げ率 > 0.5 の馬の数
```

#### 3. pace_advantage (ペース適性スコア)
```python
# この馬の脚質が予想展開に合っているか
if 予想ペース == "ハイペース":
    if 馬の脚質 == "差し" or "追い込み":
        pace_advantage = 1  # 有利
    else:
        pace_advantage = -1  # 不利
elif 予想ペース == "スローペース":
    if 馬の脚質 == "逃げ" or "先行":
        pace_advantage = 1
    else:
        pace_advantage = -1
else:
    pace_advantage = 0  # 平均ペースは中立
```

#### 4. preferred_position_rate (好走位置適合率)
```python
# 過去の好走時の序盤位置と、今回の想定位置のマッチング
過去の1-3着時の平均序盤位置 vs 今回の想定序盤位置
```

#### 5. early_speed_index (序盤スピード指数)
```python
# 過去レースの前半3Fタイムから計算
# (実際のタイムは取得困難なため、Passageの序盤位置から推定)
early_speed = 序盤位置の平均 の逆数
```

#### 6. late_speed_index (終盤スピード指数)
```python
# 既存の avg_agari を正規化・指数化

```

### 【B. セクショナル分析】(+4次元)

#### 7. agari_stability (上がり安定性)
```python
# 過去レースの上がり3Fのばらつき
std(過去の上がり3F)
```

#### 8. position_change_ability (位置取り変化能力)
```python

平均(終盤位置 - 序盤位置)
```

#### 9. distance_aptitude (距離適性)
```python

```

#### 10. track_aptitude (コース適性)
```python

同競馬場での平均着順
```
